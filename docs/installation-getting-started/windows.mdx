## Steps to Install

1. Download the Pieces for Developers <a target="_blank" href={pieces_app.links.installation.pieces_suite_manager}>Installation Manager</a> and proceed with the installation. **This will install the Pieces for Developers Desktop App and Pieces OS.**

2. Following the installation, make sure [Pieces OS](/installation-getting-started/pieces-os) is running in the background. Ensure that the Pieces logo appears in your system tray menu.

3. **Launch the Pieces for Developers Desktop App.**

:::info

If you've already installed the Pieces Desktop App, and it's having trouble connecting to [Pieces OS](/installation-getting-started/pieces-os), try these quick <a target="_blank" href={pieces_app.links.internal.faqs}>Troubleshooting Solutions</a>.

:::

## How to Update

To update the desktop installation of Pieces for Developers through the Pieces Suite installer, you can open the installer application and select the "check for updates" button in the corresponding application.

Each application will check for its own updates and begin the update if one is found. You can follow the installation progress bar to see where your installation is at in the process.

## Uninstalling Pieces for Developers
To uninstall Pieces OS and the Pieces desktop app:

1. Close both applications
2. Navigate to Settings > Apps > Installed Apps and locate Pieces/Pieces OS
3. Click on the three-dot dropdown menu and select "Uninstall"
